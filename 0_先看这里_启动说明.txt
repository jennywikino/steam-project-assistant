Steam 项目初筛助手 - 启动说明

普通 Windows 用户推荐下载：
SteamProjectAssistant-v0.9.2-portable.zip

便携版不需要自己安装 Python。

第一次使用：
1. 双击【1_首次使用_安装运行依赖.bat】
2. 双击【2_启动工具.bat】

日常使用：
- 只需要双击【2_启动工具.bat】

如果要使用 Steam 页面采集：
- 通常不用先运行【3_可选_安装Steam页面采集环境.bat】
- 工具会优先使用本机 Microsoft Edge
- 只有页面提示 Edge 不可用，或你需要完全便携 Chromium 时再运行
- 这一步会安装 Playwright Chromium，可能需要 5-15 分钟
- Chromium 会安装到当前工具包内的 app/ms-playwright 目录

不要点击：
- docs / modules / templates / tools
- app.py
- data / logs / exports / reports
这些是程序文件或运行数据，不是启动入口。

如果启动失败：
- 查看 logs 文件夹里的日志
- 或用 PowerShell 手动运行：
  python -m streamlit run app.py --server.port 8501

退出工具：
- 关闭【2_启动工具.bat】打开的黑色窗口，或在窗口里按 Ctrl+C。
- 只关闭浏览器网页不会一定停止后台服务。
- 如果 8501 端口被占用，请重新打开启动窗口并按 Ctrl+C；便携版还会提供【9_可选_强制停止工具.bat】。

卸载和清理：
- 本工具主要数据保存在解压目录内。
- 如果使用 Steam 页面采集，Chromium 会安装到 app/ms-playwright。
- 卸载方式：先关闭工具窗口，再删除整个解压目录。
- 如需清理本工具缓存，可运行【10_可选_清理本工具缓存.bat】。
