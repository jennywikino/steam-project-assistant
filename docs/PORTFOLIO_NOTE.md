# 作品集说明：Steam 项目初筛助手

## 背景

独立游戏发行日常需要大量浏览 Steam、SteamDB、活动页和候选表。项目发现通常伴随 AppID 记录、开发商/发行商确认、Demo 与简中状态检查、评论与标签整理，以及后续候选表维护。

## 我的目标

把重复的项目录入、字段补全、候选分流、日报导出做成本地工作台，让发行初筛从分散页面和手工表格，收口为一条可重复执行的工作流。

## 业务价值

- 降低手动整理成本
- 加快项目初筛速度
- 沉淀发行判断口径
- 让候选项目可追踪、可导出、可复盘

## 技术实现

- Streamlit 本地 UI
- CSV 轻量数据层
- Steam appdetails 缓存补全
- Playwright 浏览器采集
- pandas 表格处理
- openpyxl / xlsx 导出

## 我负责的能力

- 需求拆解
- 工作流设计
- 字段结构设计
- 发行判断规则设计
- 工具验收与迭代
- GitHub 版本管理

## 不做什么

- 不绕 SteamDB 反爬
- 不做全量 Steam 数据库
- 不自动替人判断签约
- 不包含公司敏感数据
