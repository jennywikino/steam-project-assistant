# NEXT_STEPS

## V0.6.0 先验收

1. 首页卡片显示好评率、评测数和评测摘要。
2. 首页卡片显示“评测样本”中位/平均游玩时间。
3. 好评率、评测数、游玩时间筛选有效；默认不误删未获取字段。
4. 首页点击项目画像后，一键项目画像能带入开发商、发行商、类型、价格和评价数据。
5. 项目画像页默认不显示“完整中国区机会”“完整竞争点分析”“完整风险与人工确认清单”。
6. 项目画像页显示紧凑发行判断卡。
7. 首页显示常用工具小方块。
8. `data/cache/steam_review_stats_cache.json` 能按 AppID 写入评价缓存。
9. Excel `首页Steam图文源` sheet 包含新增评价字段。
10. `python -m py_compile app.py modules/steam_review_stats.py modules/steam_data_normalizer.py` 通过。
11. Streamlit 能正常启动。
