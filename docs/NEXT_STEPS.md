# NEXT_STEPS

## V0.7.3 先验收

1. 打开工具。
2. 进入“Steam 浏览器采集”。
3. 点击“打开 Steam 浏览器”。
4. 外部 Chromium 窗口打开 Steam。
5. 在浏览器里打开 Steam Coming Soon / Next Fest / 搜索结果页。
6. 点击“抓取当前页 AppID”。
7. 表格出现当前页中的多个 AppID。
8. 点击“补全基础信息”。
9. 能看到开发商、发行商、发售状态、Demo、简中、类型。
10. 筛选“有 Demo / 未发售 / 自发行 / 简中 / 待试玩”可用。
11. 选中 1–3 个项目发送到候选池。
12. 候选池出现对应项目。
13. 重复发送不会重复新增。
14. SteamDB 导入、候选池、Excel 下载不受影响。
15. 如果 Playwright 未安装，页面显示清晰提示，不崩溃。
16. SteamDB Recent Events 因 403 不再作为新项目发现主入口。
17. Coming Soon / TBA 专项池后续再做。
