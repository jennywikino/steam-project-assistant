# 用户指南：Steam 项目初筛助手

## 如何启动

展示或日常使用推荐在项目目录双击 `启动工具_无黑窗.vbs`，等待浏览器打开 `http://localhost:8501`。

如果需要查看启动过程，可双击 `启动工具.bat`。

如果需要命令行启动，可以运行：

```bat
python -m streamlit run app.py --server.port 8501
```

停止工具时，关闭浏览器标签页不等于关闭后台服务。需要停止后台服务时，运行 `停止工具.bat`。

## 如何用查查

1. 打开顶部导航的“查查”。
2. 输入 Steam 链接、AppID 或游戏名。
3. 查看基础信息、Steam 入口和可用字段。
4. 需要继续处理时，可发送到项目画像或候选池相关流程。

## 如何用项目导入

1. 打开顶部导航的“项目导入”。
2. 在“导入方式”中选择 Steam 搜索导入、Steam 页面采集或 SteamDB 粘贴导入。
3. 默认推荐使用“Steam 搜索导入”。

## 如何用 Steam 页面采集

1. 打开“项目导入”，选择“Steam 页面采集”。
2. 点击“打开 Steam 浏览器”。
3. 在打开的浏览器里访问 Steam 首页、搜索页、Coming Soon、活动页等页面。
4. 回到工具页点击“抓取当前页 AppID”。
5. 对采集结果补全基础信息。
6. 勾选需要跟进的项目并发送到候选池。
7. 采集记录较多时，页面默认只展示前 100 条，可点击“显示更多 100 条”或展开完整调试表。
8. “清空本次采集”只清空 `data/steam_browser_collected.csv`，不影响候选池；清空前会自动备份。

## 如何用 Steam 搜索导入

1. 打开“项目导入”，选择“Steam 搜索导入”。
2. 选择 Coming Soon 游戏、Demo 池或自定义 Steam 搜索 URL。
3. 保持默认保守数量，或按需要选择更高数量。
4. 点击抓取后查看导入表。
5. 补全基础信息并筛选候选。
6. 发送选中项到候选池。
7. “清空本次导入”只清空 `data/steam_search_imports.csv`，不影响候选池；清空前会自动备份。
8. “按 AppID 去重”默认保留最新一条，并显示 before / after / removed。

## 如何用 SteamDB 粘贴导入

1. 打开“项目导入”，选择“SteamDB 粘贴导入”。
2. 从 SteamDB 榜单或 App 页面复制文本、链接或 AppID。
3. 粘贴到输入框并解析。
4. 检查解析结果。
5. 需要时补全 Steam 基础信息。
6. 发送选中项到候选池。

## 如何补全基础信息

在“项目导入”页的 Steam 页面采集、Steam 搜索导入或 SteamDB 粘贴导入模式中，使用现有“补全基础信息”入口。工具会基于 AppID 查询 Steam 公开基础信息，并写入开发商、发行商、发售状态、Demo、简中、类型、评论等字段。

## 如何发送候选池

1. 在导入结果表中先筛选项目。
2. 勾选要跟进的记录。
3. 点击“发送选中项到候选池”。
4. 重复 AppID 不会重复新增，会尽量补齐空字段。

## 如何导出 Excel

在“竞品与候选”或“历史与导出”中使用候选池和日报导出入口。导出结果用于人工复核、工作日报或后续候选管理。

## 常见问题

### Playwright 浏览器打不开怎么办

先确认依赖已安装，并重新启动工具。如果仍打不开，使用查查、Steam 搜索导入或 SteamDB 粘贴导入作为替代路径。

### Steam Access Denied 怎么办

这通常是 Steam 页面访问限制或网络状态导致。不要频繁重试高数量导入，先降低导入数量，或改用浏览器人工打开页面后再采集当前页 AppID。

### 为什么默认只导入 50

默认数量偏保守，是为了减少噪音数据、降低页面访问压力，并让初筛结果更容易人工复核。需要扩大范围时再手动调高。

### 为什么不自动抓 SteamDB

工具定位是本地发行初筛工作台，不做 SteamDB 自动抓取，也不绕反爬。SteamDB 相关内容通过手动粘贴导入，保留人工确认边界。

### 数据文件在哪里

主要本地数据在 `data/` 目录，常见文件包括 `data/candidate_pool.csv`、`data/steam_browser_collected.csv` 和 `data/steam_search_imports.csv`。缓存通常在 `data/cache/`。

### 如何清空采集表

进入“项目导入”页面，选择“Steam 页面采集”，使用页面内的清空入口。清空前请确认当前采集结果不再需要；该操作只清理采集表，不会修改候选池。

### 如何清空 Steam 搜索导入表

进入“项目导入”页面，选择“Steam 搜索导入”，勾选“我确认清空本次导入”后点击“清空本次导入”。该操作只清空当前导入列表，不会修改候选池。
