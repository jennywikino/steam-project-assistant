# 作品集截图说明

本目录保存 Steam Project Assistant 的脱敏展示截图，仅用于 README 和项目介绍。

| 文件 | 用途 |
|---|---|
| `01_home_feed.png` | 首页项目发现 Feed，展示首页工作台、数据概览和项目卡片 |
| `03_import.png` | 项目导入 / Steam 源筛选，展示项目来源选择与导入流程 |
| `04_project_profile.png` | 一键项目画像，展示单项目基础信息、评论样本和公告记录 |
| `05_candidate_pool.png` | 候选池工作台，展示筛选、优先级、处理阶段和下一步动作 |

## 截图要求

- 使用统一窗口尺寸和缩放比例。
- 隐藏用户名、本地路径、真实备注和联系方式。
- 展示数据应经过脱敏，不包含真实业务信息。
- 不提交 `data/cache`、`exports`、`reports`、`debug` 中的真实数据。
