# 作品集截图说明

本目录用于保存 Steam Project Assistant 的脱敏展示截图。截图应使用 Demo 数据或确认可公开的数据，不包含真实候选池、缓存、导出报告或调试信息。

## 截图清单

| 顺序 | 文件名 | 展示内容 |
| --- | --- | --- |
| 01 | `01_home_feed.png` | 首页工作台、项目发现 Feed、数据概览和快捷入口 |
| 02 | `02_import.png` | Steam 搜索、页面采集或 SteamDB 粘贴解析入口 |
| 03 | `03_candidate_pool.png` | 候选池筛选、优先级和处理阶段 |
| 04 | `04_enrich_queue.png` | Steam 基础资料补全队列和处理结果 |
| 05 | `05_project_profile.png` | 单项目画像、基础信息和辅助样本 |
| 06 | `06_export.png` | CSV、Excel 或 TXT 下载区域 |

## 截图要求

- 使用统一窗口尺寸和缩放比例。
- 隐藏用户名、本地路径、真实备注和联系方式。
- 不提交 `data/cache`、`exports`、`reports`、`debug` 中的真实数据。
- PNG 文件准备完成前，仅保留本说明文件，不创建空白或伪造图片。
